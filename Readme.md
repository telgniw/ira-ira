Water Ira-Ira Bou
=================

Commands
--------

    # import/export settings
    /_> import testcases/VIDEO001.txt
    /_> export testcases/VIDEO001.txt

    # set video/camera source
    /_> open testcases/VIDEO001.txt
    /_> open 0

    # modify settings
    /_> set area
    /_> set check
    /_> set color
    /_> set start

    # ui instructions for settings
    y to select point
    s to save
    q to cancel

    # start running the program
    /_> start
